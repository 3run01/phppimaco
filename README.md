O PHP PIMACO é um pacote para geração de etiquetas que tem o intuito de auxiliar a montagem de PDFs com as tuas etiquetas devidamente formatadas.

## Instalação

Para fazer instalação do PHPPimaco utilize o composer
```php
composer require proner/phppimaco
```
ou adicione isso ao require do seu composer.json
```php
"require":{
    "proner/phppimaco": "dev-master"
}
```
